# DevMind

DevMind is an OpenSpec-first repository for a Spring Boot backend, a Python AI worker, and a Vue frontend.

## Start Here

- Canonical change tracking: `openspec/changes/`
- Active foundation change: `openspec/changes/establish-project-foundation/`
- Contributor rules: `AGENTS.md`
- Product context: `devmind_project_spec.md`

## Module Entry Points

- `devmind-backend/` - Spring Boot backend
- `devmind-ai-worker/` - Python and FastAPI worker
- `devmind-frontend/` - Vue frontend
- `deploy/` - deployment assets

## Workflow

1. Inspect the active change under `openspec/changes/<change-name>/`
2. Use `tasks.md` as the only canonical task list
3. Keep specs, code, and tests aligned before declaring work complete
